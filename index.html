Custom Transformer

This project implements a lightweight Transformer-based language model in PyTorch, inspired by the GPT architecture. It features key components like multi-head self-attention, positional embeddings, and stacked Transformer blocks to capture long-range dependencies in text data. The model is trained on custom text inputs for sequence-to-sequence learning, leveraging a highly efficient training loop and optimization techniques such as layer normalization and dropout. Designed with a focus on simplicity and scalability, this project demonstrates how modern neural networks can generate coherent text sequences while providing a foundation for deeper exploration into natural language processing.

  • Date: January 2022
  • Category: AI

CNN

This project showcases a convolutional neural network (CNN) for handwritten digit classification on the MNIST dataset using PyTorch. The model consists of convolutional layers with ReLU activations, dropout for regularization, and fully connected layers for classification into ten digit categories. It utilizes data loaders for efficient training and testing, employing the Adam optimizer and cross-entropy loss for optimization. The model is trained for multiple epochs, achieving high accuracy on the test set, and includes a feature to visualize predictions for individual test images, demonstrating the power of CNNs in computer vision tasks.

  • Date: January 2023
  • Category: AI

Custom Transformer for Text Translation

This project implements a Transformer model for neural machine translation, converting English numbers to their French equivalents. It features self-attention mechanisms and multi-head attention layers to capture contextual relationships between words. The model comprises an Encoder-Decoder architecture, with position embeddings and layer normalization for stability and performance. Training is conducted using tokenized, padded sequences and a cross-entropy loss function. The system can process input sentences, output translations, and achieves accurate results on small-scale test data, demonstrating the effectiveness of the Transformer architecture for sequence-to-sequence tasks.

  • Date: January 2023
  • Category: AI

Image Detection Using YOLOv5

project demonstrates the use of YOLOv5, a state-of-the-art deep learning model for object detection. The process begins by downloading the YOLOv5 repository from GitHub and installing the required dependencies. The YOLOv5s model, a lightweight version, is then loaded using PyTorch. An image is selected for processing, where the model identifies and detects objects within it. Finally, the results are visualized using Matplotlib, showcasing the detected objects directly on the image. This project highlights the efficiency and accuracy of YOLOv5 in object detection tasks.
